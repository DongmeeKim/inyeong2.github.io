#### **문제**
함수를 완성해서 매개변수 list의 평균값을 return하도록 만들어 보세요. 어떠한 크기의 list가 와도 평균값을 구할 수 있어야 합니다.

- - -
#### **풀이**(171205 ver.)
```
def average(list):
    s = sum(list) / len(list)
    return s

# 아래는 테스트로 출력해 보기 위한 코드입니다.
list = [5,3,4]
print("평균값 : {}".format(average(list)))
```
